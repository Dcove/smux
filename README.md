# SMUX
Termux下的一个多功能脚本
-------
## 安装  
对于smux，你可以使用如下命令来安装  
`git clone https://github.com/Dcove/smux.git`  
在此步骤完成之后，你可以执行`install.sh`以安装smux或执行`bash smux`以临时启动smux。
------
## 功能
* 切换termux为清华大学源
* 一键安装zsh和配置Oh-My-ZSH
* 全新的smuxsh
------
